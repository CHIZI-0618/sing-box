## 额外功能

### Yacd面板支持重载配置文件

```
警告：
1. 重载配置文件会使得面板重启，在重启过程中面板可能会无响应，请耐心等待
2. 请确保配置文件正确，否则可能会导致程序无法启动
3. 若配置文件中有proxy-provider，重载配置文件后，proxy-provider可能会更新，更新缓慢会导致启动缓慢，请耐心等待
```
